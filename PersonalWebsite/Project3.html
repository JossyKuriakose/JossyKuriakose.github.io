This project implements Kruskal's Algorithm to find the Minimum Spanning Tree (MST) of a weighted, undirected graph. A Minimum Spanning Tree connects all the vertices in a graph with the least total edge weight and no cycles. The algorithm sorts all edges by weight and adds them one by one, ensuring that no cycles are formed, until all vertices are connected.
